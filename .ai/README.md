# `./.ai`

## Proposito

Memoria local de esta carpeta para `dash-noc`.

Esta carpeta existe para poder retomar el contexto sin depender solo de `/ai/history`.

## Contexto principal

- La carpeta actual es `/root/projects/dash-noc`.
- El proyecto antiguo recuperado de la memoria historica esta en `/projects/proxmox-thermals`.
- `dash-noc` es una pantalla NOC para tablet servida desde Proxmox.
- `proxmox-thermals` queda como fuente/referencia, no como alcance completo.

## Documentos

- `dash-noc.md`: resumen operativo para agentes IA.
- `thermals-front.md`: resumen funcional y tecnico de la pagina termica heredada.
- `history/`: bitacora cronologica de lo documentado o cambiado en esta carpeta.

## Criterio de mantenimiento

- Mantener aqui solo informacion util para agentes IA.
- No guardar secretos, credenciales ni datos sensibles de VMs.
- Si se copia o mueve codigo desde `/projects/proxmox-thermals`, registrar la decision en `history/`.
